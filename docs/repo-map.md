# Repo Map

- `plugin/` — JUCE plugin shell
- `backend/` — FastAPI service and runners
- `tester/` — browser-based prompt/test frontend
- `shared/` — schemas and voice registries
- `scripts/` — helper scripts for development and linkage
- `docs/` — architecture, packaging, attribution, SEO, roadmap
- `third_party/` — optional PersonaPlex checkout or submodule mount
